Examples

New

from agescx import *

scenario = Scenario()

will create blank scenario

Load

there's two way to load scenario.

while initializing 

from agescx import *

scenario = Scenario("myScenario.scx")

or 

from agescx import *

scenario = Scenario()
scenario.load("myScenario.scx")

Save

scenario.save("myEditedScenario.scx")
